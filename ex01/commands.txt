СПРАВОЧНИК ПО КОМАНДАМ LINUX/MACOS

===
Название: cal
Описание: Выводит календарь на месяц или год.
Синтаксис: cal [месяц] [год]
Пример:
cal          # Календарь на текущий месяц
cal 12 2023  # Календарь на декабрь 2023
cal -y       # Календарь на текущий год (Linux)
cal 2023     # Календарь на весь 2023 год (macOS)
Примечание: В macOS флаг -y не работает, вместо него нужно указывать год без месяца. В Linux оба варианта работают.
===

===
Название: factor
Описание: Разлагает число на простые множители.
Синтаксис: factor [число]
Пример:
factor 1024
# Вывод: 1024: 2 2 2 2 2 2 2 2 2 2
factor 17
# Вывод: 17: 17
===

===
Название: seq
Описание: Генерирует последовательность чисел.
Синтаксис: seq [первые] [приращение] [последнее]
Пример:
seq 5           # 1 2 3 4 5
seq 0 2 10      # 0 2 4 6 8 10
seq -f "file%.0f.txt" 1 3  # file1.txt file2.txt file3.txt
===

===
Название: sl
Описание: Анимация паровоза, проезжающего через терминал.
Синтаксис: sl [опции]
Пример:
sl      # Базовая анимация
sl -F   # Паровоз летит
sl -l   # Маленький паровоз
Примечание: Требует установки (macOS: brew install sl, Linux: sudo apt install sl).
===

===
Название: units
Описание: Конвертер единиц измерения.
Синтаксис: units [величина] [единица]
Пример:
units '100 miles' 'km'  # Конвертирует 100 миль в километры
units '1 gallon' 'liters' # Конвертирует галлоны в литры
Примечание: Запуск без аргументов переводит в интерактивный режим.
===

===
Название: curl
Описание: Передача данных по URL.
Синтаксис: curl [опции] [URL]
Пример:
curl -I https://google.com          # Заголовки ответа
curl -O https://example.com/file.zip # Скачать файл
curl -s https://api.github.com/users/octocat # JSON-ответ
===

===
Название: lsof
Описание: Показывает открытые файлы и процессы.
Синтаксис: lsof [опции]
Пример:
lsof -i :80              # Процессы, использующие порт 80
lsof -u username         # Файлы, открытые пользователем
lsof /var/log/system.log # Кто использует файл
===

===
Название: tar
Описание: Создание и распаковка архивов.
Синтаксис: tar [опции] [файл]
Пример:
tar -czvf archive.tar.gz /path/to/dir  # Создать архив
tar -tzvf archive.tar.gz               # Просмотреть архив
tar -xzvf archive.tar.gz               # Распаковать архив
Примечание: Ключи: c - создать, x - извлечь, z - gzip, v - verbose, f - файл.
===

===
Название: dig
Описание: DNS-запросы.
Синтаксис: dig [@сервер] [имя] [тип]
Пример:
dig google.com ANY          # Все записи домена
dig @8.8.8.8 google.com MX  # Запрос MX-записей через Google DNS
dig +short google.com       # Только IP-адрес
===

===
Название: rsync
Описание: Синхронизация файлов и каталогов.
Синтаксис: rsync [опции] [источник] [цель]
Пример:
rsync -avz ~/documents/ user@server:/backup/  # Синхронизация с сервером
rsync -nav --delete source/ target/           # Тестовый запуск
Примечание: Ключи: a - архивный режим, v - verbose, z - сжатие, n - тестовый режим.
===
